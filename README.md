# Excel to Image Auto Fill (UiPath RPA)

本專案為一套使用 **UiPath RPA** 製作的自動化流程，  
能夠自動完成以下工作流程：

1. 讀取 Excel 資料
2. 開啟指定圖片模板
3. 將 Excel 對應欄位資料填入圖片指定位置
4. 產出並另存為新的 PNG 圖檔

適用於：  
📌 表單影像填值  
📌 證書、文件、名牌、票證圖片自動產生  
📌 大量圖像資料自動化處理
